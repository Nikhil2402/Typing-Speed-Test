                        RULES

     1) Use of backspace key is not allowed!
     2) The timer starts running whenever the user will start typing.
     3) All letters are case sensitive.
     4) WPM is based on how much correctly you type,
	even if a single letter is incorrect, the whole word will be considered as incorrect.
     5) Space for typing is provided just below each line of the paragraph for convinence.
